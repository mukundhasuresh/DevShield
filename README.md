# DevShield — AI-Powered DevSecOps SaaS

## Project Overview
DevShield is a B2B SaaS GitHub App that automatically scans every Pull Request for security vulnerabilities using AI, posts findings as PR comments, and writes an immutable audit log to the Polygon blockchain.

## Getting Started

1. Clone the repository
2. Set up environment variables locally in a `.env.local`
3. Run `docker-compose up -d` to start Redis and the backend services.
4. Run Next.js locally from `apps/web`.
